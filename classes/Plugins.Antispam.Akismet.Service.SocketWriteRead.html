      Utility class used by Akismet

      This class is used by Akismet to do the actual sending and receiving of data. It opens a connection to a remote host, sends some data and the reads the response and makes it available to the calling program. The code that makes up this class originates in the Akismet WordPress plugin, which is available on the Akismet website. N.B. It is not necessary to call this class directly to use the Akismet class. This is included here mainly out of a sense of completeness.
