      Based off of Joomla's JObject

      Generate letter responses array for a given asset id

      getLetterResponsesForAssetId(object $db, integer $asset_id, boolean $include_headers = true, integer $section_id = NULL) : array
      Static

      A lot of the complexity of this method stems from the fact that the form data is not actually labeled, but rather merely coordinates on a page. We must therefore make assumptions on question numbers and response labels based on location within the page. This is a significant shortfall, especially given that questions or responses may at some point read horizontally, rather than vertically.
