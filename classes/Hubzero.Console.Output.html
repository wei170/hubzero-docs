      Track default indentation for lines

      $defaultIndentation : string

      Default

      ''

      If prefering predominant indentation other than 0, set with setDefaultIndentation() to avoid having to set on all calls to addLine().

       

      Track whether we're in interactive mode

      $isInteractive : string

      Default

      true

      While in interactive mode, output each line as it's given rather than pooling and waiting until render is called.

       

      Set the output mode

      $mode : string

      Default

      'normal'

      Assume normal, but minimal and verbose are also options. This isn't setting the format, but it's allowing us to distinguish what amount/sorts of data the command should return. The individual format handler would then format the output appropriately.
