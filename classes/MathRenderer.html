    Contain everything related to parsing

    Takes LaTeX fragments, sends them to a helper program (texvc) for rendering to rasterized PNG and HTML and MathML approximations. An appropriate rendering form is picked and returned.

    Based off code by Tomasz Wegrzanowski, with additions by Brion Vibber (2003, 2004)
